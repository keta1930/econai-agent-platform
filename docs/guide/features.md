# 功能总览

## 超级管理员能做什么

超级管理员是平台的最高权限角色，通常由平台部署者担任。

| 功能 | 说明 |
|------|------|
| 生成邀请码 | 创建邀请码发给教师，教师凭此注册账号 |
| 管理教师 | 查看所有教师、启用/禁用账号、重置密码、删除账号 |

## 教师能做什么

教师是平台的核心使用者，负责管理自己的班级和作业。

| 功能 | 说明 |
|------|------|
| 班级管理 | 创建班级、管理加入凭证 |
| 学生名单 | 添加预期学号、批量导入、审批密码重置 |
| 发布作业 | 设定题目、说明、评分标准（支持 AI 辅助生成） |
| 查看批改 | 查看每个学生的提交和 AI 批改结果 |
| AI 助教 | 自然语言对话，查询数据、设计作业、分析成绩 |
| 课程分享 | 创建分享主题让学生投票，管理分享材料 |
| 模型管理 | 添加、切换 AI 模型配置 |
| 数据备份 | 一键备份和恢复 |

## 学生能做什么

| 功能 | 说明 |
|------|------|
| 加入班级 | 用教师给的凭证注册并加入 |
| 提交作业 | 文本粘贴、文件上传、拍照上传三种方式 |
| 查看反馈 | AI 批改的详细反馈：分数、维度评价、建议、亮点 |
| 成绩汇总 | 查看所有作业的成绩和提交历史 |
| 课程分享 | 投票选择分享主题，查看分享材料 |
